# 🎓 Lost & Found Campus – A Smart Recovery System for University Students

> *"Bridging the gap between finders and owners, one lost item at a time."*

---

## 📖 Overview

**Lost & Found Campus** is a real-time native Android application designed specifically for the University of Frontier Technology, Bangladesh (UFTB).  
It replaces scattered social media posts and physical notice boards with a **centralized, secure, and instant** platform for reporting and recovering lost items on campus.

Whether it's an ID card, wallet, calculator, or book — this app ensures nothing stays lost for long.

---

## 🔍 The Problem We Solve

| Issue | Current Situation | Our Solution |
|-------|------------------|---------------|
| 📢 Inefficiency | Important lost/found posts get buried in WhatsApp or Facebook groups | A dedicated, searchable app timeline |
| 🧩 Fragmentation | No single source of truth for lost items | Centralized database with real-time sync |
| 🔐 Privacy Risk | Students post personal contact info publicly | In-app secure messaging & calling |

---

## 🎯 Project Objectives

1. ✅ Create a **single reliable hub** for all lost & found items on campus  
2. ⚡ Enable **real-time visibility** of newly found items  
3. 📉 **Reduce average recovery time** significantly compared to current methods

---

## ✨ Key Features

- 🔒 **Secure Authentication** – Signup/login via Firebase Auth (verified access only)  
- 📸 **Instant Multimedia Posting** – Upload photos, descriptions, and location (Cafeteria, Library, etc.)  
- 🔍 **Smart Search & Filters** – Search by keywords or filter by category  
- 💬 **Direct Contact System** – In-app messaging/calling (no phone numbers exposed publicly)  
- ✅ **Post Status Management** – Mark as "Solved" to archive and keep feed clean  

---

## 👥 Target Users & Use Case

**Who:** Students, Faculty, Campus Security Staff  

**📱 Example Scenario:**  
> A student finds a wallet in the Central Library. They snap a photo, add a description, and post it.  
> The owner receives an instant alert, contacts the finder via the app, and arranges a safe campus meeting point to retrieve it.

## 🛠️ Tech Stack

| Component | Technology | Why? |
|-----------|------------|------|
| Frontend | Android Studio (Kotlin + Jetpack Compose) | Industry standard for high-performance native Android apps |
| Backend & Database | Firebase Cloud Firestore | Real-time sync — updates appear instantly for all users |
| Storage | Firebase Storage | Secure hosting for item images |
| Auth | Firebase Authentication | Easy, secure, verified access |

---

## 📱 Platform Choice: Native Android

**Justification:**  
Native development gives full access to hardware features (camera, push notifications) and delivers the best UX for UFTB students.
